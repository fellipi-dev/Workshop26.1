# Projeto de Machine Learning - Classificação

## Objetivo
Criar um modelo de classificação para prever o tipo de motor (engine_type) de veículos da BMW.

## Base de dados
BMW Global Sales Dataset

## Etapas realizadas
- Importação dos dados (CSV)
- Exploração dos dados
- Tratamento dos dados
- Separação em treino e teste
- Treinamento do modelo (Random Forest)
- Avaliação do modelo

## Resultado
Acurácia: 100%

## Ferramenta utilizada
Google Colab
